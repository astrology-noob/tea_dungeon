Динамические связи отношений  
1) accoциация - "я знаю о" ("я использую")  
2) агрегация - "я состою из"  
3) композиция - "я состою из и не могу без этого существовать"  
```c#
class A {  
    private B b = new B;
}
```

\+ самоассоциация
```c#
class B {  
    private B b;
}
```
